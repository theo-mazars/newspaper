# Newspaper

## Getting Started

This is how to run the dev server:

```bash
yarn dev
```

Open [http://localhost:4001](http://localhost:4001) with your browser to see the result.

## Technologies used
- NextJS (FullStack)
- PWA (For mobile scalability)
- Google News RSS (getting datas)
